# Smart Robotics Solutions for an Industrial Production Line  
### Pick-and-Sort Using NED2 Robot

## 📌 Overview
This project focuses on designing and simulating a smart robotic system for industrial production lines using the NED2 robotic arm. The system performs pick-and-place and pick-and-sort tasks autonomously within a simulated environment.

The project demonstrates how robotics can improve efficiency, accuracy, and flexibility in modern manufacturing systems.

---

## 🎯 Objective
To design and implement a robotic system capable of performing automated pick-and-place tasks using the NED2 robot within a simulated production line.

---

## 🤖 System Description
The system simulates an industrial production line where a robotic arm interacts with objects moving on a conveyor belt. The robot detects, picks, and places objects accurately based on predefined logic.

---

## ⚙️ Robot Specifications
- Robot: NED2 Collaborative Robotic Arm  
- Degrees of Freedom: 6 DOF  
- End Effector: Gripper  
- Motion Type: Joint and Cartesian movement  
- Environment: Simulation-based  

---

## 🛠️ Software and Tools
- ROS2 (Robot Operating System) for robot control, communication, and motion planning  
- Niryo Studio & Niryo Academy for robot configuration and programming  
- Simulation environment for testing and validation  

---

## 🔄 System Workflow
- Objects move through a conveyor belt  
- Vision system detects object position and type  
- Robot selects appropriate tool (gripper, vacuum, electromagnet)  
- Robot performs pick-and-place or pick-and-sort task  
- Objects are placed in the correct location  

---

## 🧠 Key Features
- Autonomous robot control using ROS2  
- Pick-and-place and pick-and-sort operations  
- Multi-tool selection (gripper, vacuum, electromagnet)  
- Vision-based object detection  
- Simulation-based testing for safe validation
